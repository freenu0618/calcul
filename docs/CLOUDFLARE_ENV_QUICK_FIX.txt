==================================================
Cloudflare Pages 환경변수 빠른 수정 가이드
==================================================

1단계: Cloudflare Pages 대시보드 접속
----------------------------------------
https://dash.cloudflare.com
→ Pages 클릭
→ calcul (프로젝트 선택)
→ Settings 탭
→ Environment variables 클릭


2단계: Production 환경변수 수정
----------------------------------------
1. Production 탭 선택
2. VITE_API_BASE_URL 찾기
3. Edit (연필 아이콘) 클릭
4. 아래 값을 복사해서 붙여넣기:

https://paytools.work

⚠️ 주의사항:
- 따옴표("") 없이 붙여넣기
- 끝에 슬래시(/) 없음
- 콤마(,) 없음

5. Save 버튼 클릭


3단계: 재배포
----------------------------------------
1. Deployments 탭으로 이동
2. 최신 배포 찾기 (커밋 3775b11 또는 8ce435d)
3. 오른쪽 ... 버튼 클릭
4. "Retry deployment" 선택
5. 빌드 완료 대기 (약 2-3분)


4단계: 검증
----------------------------------------
배포 완료 후 브라우저에서:

https://calcul-1b9.pages.dev
또는
https://paytools.work

접속 → 회원가입/로그인 → 정상 작동 확인


==================================================
FAQ: 왜 하나만 설정하나요?
==================================================

Q: 두 개 도메인이 있는데 왜 하나만 설정하나요?

A: VITE_API_BASE_URL은 "백엔드 API 서버 주소"입니다.
   백엔드는 하나만 있습니다: https://paytools.work

   두 프론트엔드 도메인:
   - https://calcul-1b9.pages.dev ✅
   - https://paytools.work ✅

   백엔드 API:
   - https://paytools.work/api/v1 ✅ (하나만!)

   두 도메인 모두 같은 백엔드에 연결됩니다.


Q: CORS는 어떻게 되나요?

A: 백엔드의 ALLOWED_ORIGINS 설정이 두 도메인 모두 허용합니다.
   (이미 설정되어 있음)

   ALLOWED_ORIGINS=https://calcul-1b9.pages.dev,https://paytools.work

   이것은 백엔드 설정이므로 건드릴 필요 없습니다.
