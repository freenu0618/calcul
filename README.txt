                         급여 계산기 - 사용자 가이드
================================================================================

안녕하세요! 급여 계산기를 이용해 주셔서 감사합니다.
아래 안내에 따라 프로그램을 설치하고 실행해 주세요.


================================================================================
                            1. Docker 설치하기
================================================================================

이 프로그램은 Docker를 사용합니다. 
아래 링크에서 Docker Desktop을 다운로드해 주세요.

▶ 다운로드 링크:
   https://www.docker.com/products/docker-desktop/

[설치 방법]
1. 위 링크에서 "Download for Windows" 버튼 클릭
2. 다운로드된 "Docker Desktop Installer.exe" 실행
3. 설치 화면에서 "Install" 클릭
4. 설치 완료 후 컴퓨터 재시작
5. Docker Desktop 프로그램 실행 (바탕화면 아이콘 또는 시작메뉴)
6. Docker가 실행되면 우측 하단 트레이에 고래 아이콘이 나타납니다


================================================================================
                           2. 프로그램 파일 확인
================================================================================

프로그램 폴더에 다음 파일/폴더가 있는지 확인해 주세요:

📁 급여계산기/
   ├── 📄 docker-compose.yml    ← Docker 설정 파일 (필수)
   ├── 📄 README.txt            ← 지금 보고 계신 파일
   ├── 📁 backend/              ← 서버 프로그램 (필수)
   └── 📁 frontend/             ← 웹 화면 프로그램 (필수)


================================================================================
                           3. 프로그램 시작하기
================================================================================

[방법 1: 명령 프롬프트 사용]

1. 키보드에서 Win + R 키를 누릅니다
2. "cmd" 입력 후 Enter
3. 아래 명령어를 순서대로 입력합니다:

   cd 프로그램폴더경로
   docker-compose up -d

   예시:
   cd C:\Users\사용자이름\Downloads\급여계산기
   docker-compose up -d

4. "Started" 메시지가 나오면 성공!
5. 웹 브라우저를 열고 아래 주소로 접속합니다:

   ▶ http://localhost:3000


[방법 2: PowerShell 사용]

1. 프로그램 폴더에서 빈 공간을 Shift + 마우스 우클릭
2. "여기에서 PowerShell 창 열기" 선택
3. 아래 명령어 입력:

   docker-compose up -d

4. 웹 브라우저에서 http://localhost:3000 접속


================================================================================
                           4. 프로그램 종료하기
================================================================================

프로그램을 종료하려면:

1. 명령 프롬프트 또는 PowerShell 열기
2. 프로그램 폴더로 이동
3. 아래 명령어 입력:

   docker-compose down

4. "Removed" 메시지가 나오면 종료 완료


================================================================================
                              5. 문제 해결
================================================================================

❌ "docker-compose" 명령이 인식되지 않을 때
   → Docker Desktop이 실행 중인지 확인하세요
   → 컴퓨터를 재시작한 후 다시 시도하세요

❌ 웹페이지가 열리지 않을 때
   → 잠시 기다린 후 다시 접속해 보세요 (약 10초)
   → 주소가 http://localhost:3000 이 맞는지 확인하세요

❌ "포트 3000이 사용 중입니다" 오류
   → 다른 프로그램이 해당 포트를 사용 중입니다
   → 해당 프로그램을 종료하거나, docker-compose.yml에서 포트 변경


================================================================================
                              6. 자주 묻는 질문
================================================================================

Q: Docker는 무료인가요?
A: 네, 개인 사용자와 소규모 기업은 무료로 사용할 수 있습니다.

Q: 인터넷 연결이 필요한가요?
A: 최초 실행 시에만 필요하며, 이후에는 오프라인에서도 사용 가능합니다.

Q: 데이터는 어디에 저장되나요?
A: 프로그램 폴더 내 data/ 폴더에 저장됩니다.
